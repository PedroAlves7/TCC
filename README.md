The thesis proposes a development of a diferential robot and a way to evaluate and compare mapping accuracy SLAM algorithms.

# Link list

The ROS repository is available at [PedroAlves7/my_bot](https://github.com/PedroAlves7/my_bot).

The LIDAR is available at [PedroAlves7/cspc_lidar](https://github.com/PedroAlves7/cspc_lidar).

The comparison was done by lining up the maps with ICP and calculating the score based on the accuracy of the mapping. The script for comparison can be founded in this repository.
